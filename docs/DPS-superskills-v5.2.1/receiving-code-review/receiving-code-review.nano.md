# receiving-code-review — nano

**Trigger:** When receiving code review feedback.

**Non-negotiable:**
- Clarify ALL unclear items before implementing anything.
- Verify feedback against codebase before implementing. Never blind implementation.
- No performative agreement ("You're absolutely right!"). Actions speak.

**Output:** All items resolved with technical reasoning. tdd-verified each fix.
