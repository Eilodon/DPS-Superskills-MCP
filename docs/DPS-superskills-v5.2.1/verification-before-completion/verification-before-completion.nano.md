# verification-before-completion — nano

**Trigger:** Before saying done/fixed/passing/complete.

**Iron law:** no completion claim without fresh T1/T2 evidence. T4 is never evidence.

Use claim grammar: CLAIM / EVIDENCE / SOURCE / SCOPE / RESIDUAL RISK. Subagent reports are T3 until parent verifies raw output/artifact. For bug fixes, include Fix Anchor and run pattern-globalize.
