# Super Skills

This project uses the Super Skills methodology (Superskills × VHEATM).

## MANDATORY: Before any task

Load the `using-super-skills` skill before any response or action.
If a skill might apply — even 1% chance — invoke it first.

## Skill invocation (Codex)

```
/plugins → search "super-skills" → Install
skill using-super-skills
```

Or reference skills directory directly: `~/.agents/skills/`

## Priority

1. Instructions in this file (AGENTS.md)
2. Super Skills methodology
3. Default behavior
